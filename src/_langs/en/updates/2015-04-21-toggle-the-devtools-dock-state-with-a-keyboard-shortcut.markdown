Use <kbd class="kbd">Cmd + Shift + D</kbd> to toggle the DevTools dock state between the last two recently used states. Dock options are:

<ul>
<li>Dock to right</li>
<li>Dock to bottom</li>
<li>Undock DevTools</li>
</ul>
