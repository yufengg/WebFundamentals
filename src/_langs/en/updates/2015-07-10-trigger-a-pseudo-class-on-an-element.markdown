You can trigger pseudo classes through DevTools, for example:

<ul>
<li>Hover</li>
<li>Active</li>
<li>Focus</li>
<li>Visited</li>
</ul>

Also note, when you do this, a small orange indicator appears to the left of the node in the DOM tree so you know that its state has changed.
