最终
使所有列都垂直排列。  为此布局模式
选择视图断点时要依据内容，并且将针对每种设计
而改变。

与 Mostly fluid 示例类似，内容在最小视图中垂直
排列，但当屏幕宽度超过 600px 时，第一个和第二个内容
`div` 占用屏幕的全宽度。  `div` 的顺序是使用
 CSS 属性的顺序进行设置的。  在 800px 时，
使用全屏宽度来显示全部三个内容 `div`。

使用此模式的网站包括：

 * [Modernizr](http://modernizr.com/)
 * [Wee Nudge](http://weenudge.com/)
