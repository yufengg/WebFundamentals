最後結果是所有欄會被垂直堆疊。
  為這種版面配置模式選擇中斷點，
視內容而定，也會因每一種設計而變動。

就像主體為流動的範例，
內容被垂直堆疊於最小檢視中，但在螢幕擴展超出 600px 時，主要與次要內容 
`div` 會佔據整個螢幕寬度。  `div` 的順序是以舊 CSS 屬性來設定。
  在 800px 時，所有三個內容 `div` 都會顯示，使用完整螢幕寬度。


使用這種模式的網站包括：

 * [Modernizr](http://modernizr.com/)
 * [Wee Nudge](http://weenudge.com/)
